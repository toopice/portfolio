Titanic (sql)

Durant ce projet, j'ai travaillé en équipe de deux personnes. Notre objectif était de recréer une base de données sur le Titanic. Ce projet impliquait plusieurs étapes,

Collecte de données : Nous avons recherché et rassemblé diverses sources d'information sur les passagers du Titanic, y compris leurs noms, âges, classes, genres, tarifs payés, et le sort qu'ils ont connu.

Création de la base de données : Nous avons utilisé un système de gestion de base de données relationnelle (SGBDR) tel PostgreSQL pour structurer et stocker les données. Cela a impliqué la conception du schéma de la base de données, y compris la définition des tables, des champs et des relations entre eux.
